# Librarian Books Preview

Fabric mod for **Minecraft 1.21.11** that previews **librarian villager** enchanted book trades while you aim at them.

## Features

- Shows up to **3** enchanted book offers above the targeted librarian.
- Displays the enchantment name using the player's language (Minecraft translations).
- Displays the emerald cost (uses the trade's displayed buy items).
- HUD overlay includes **icons** (enchanted book + emerald) and localized pluralization for emeralds.

## Requirements

- Minecraft **1.21.11**
- Fabric Loader
- Fabric API

## Install

1. Install Fabric Loader for Minecraft 1.21.11.
2. Install Fabric API.
3. Put the mod `.jar` into your Minecraft `mods` folder.

## How to use

1. Find a **librarian villager**.
2. Aim your crosshair at the villager.
3. The overlay shows the enchanted book trades and emerald prices.

# Librarian Books Preview (Português)

Mod Fabric para **Minecraft 1.21.11** que mostra uma prévia dos **livros encantados** vendidos por **aldeões bibliotecários** enquanto você mira neles.

## Recursos

- Mostra até **3** ofertas de livros encantados acima do aldeão mirado.
- Nome do encantamento no idioma do jogador (traduções do Minecraft).
- Preço em esmeraldas baseado nos itens exibidos no trade.
- Overlay com **ícones** (livro encantado + esmeralda) e pluralização localizada de esmeraldas.

## Requisitos

- Minecraft **1.21.11**
- Fabric Loader
- Fabric API

## Instalação

1. Instale o Fabric Loader para Minecraft 1.21.11.
2. Instale a Fabric API.
3. Coloque o arquivo `.jar` do mod na pasta `mods`.

## Como usar

1. Encontre um **aldeão bibliotecário**.
2. Mire nele com a mira.
3. O overlay vai mostrar as ofertas de livros encantados e o preço em esmeraldas.

## Observações / Limitações

- No Minecraft 1.21.11, as ofertas do aldeão não podem ser lidas no client; este mod consulta as trades no servidor e envia a prévia para o client.
- A prévia foca nas ofertas de livro encantado e no custo em esmeraldas.
